# My-Defuddle (Enhanced Web & YouTube Extractor)

這是一個基於 `defuddle` 但針對「YouTube 影片」進行深度優化的進階技能。

## 核心升級：
1. **YouTube 智能增強**：
   - 當輸入 YouTube 連結時，除了提取原始逐字稿，會自動執行 **繁體中文重點分析**。
   - 自動生成相關的 **Hashtags** (標籤)。
   - 自動加上當天日期標籤 `#yyyy-mm-dd` 以便在 Obsidian 中進行日誌整理。
2. **網頁去雜質**：
   - 繼續維持原版 `defuddle` 的高效 Markdown 提取能力。
3. **繁體中文優先**：
   - 所有的總結與分析預設為繁體中文輸出。

## 使用方式：
`/my-defuddle <URL>`
