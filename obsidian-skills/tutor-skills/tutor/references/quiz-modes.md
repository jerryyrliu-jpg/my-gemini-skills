# Quiz Modes

## 1. Diagnostic (診斷)
- Target: All un-assessed concepts.
- Purpose: Set baseline mastery.

## 2. Drill (弱項強化)
- Target: Concepts marked with 🟥 or 🟨.
- Focus: High-intensity testing on weak spots.

## 3. Choose Section (按需學習)
- Target: Specific MOC or topic chosen by user.

## 4. Hard-mode (挑戰模式)
- Target: Mastery 🟩 or 🟦.
- Strategy: Re-phrased questions in different contexts to ensure deep understanding.
