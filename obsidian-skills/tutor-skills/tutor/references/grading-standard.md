# Grading Standard

## Criteria
- **🟥 (Failed)**: Fundamental misunderstanding.
- **🟨 (Partial)**: Correct core idea but missed details.
- **🟩 (Mastered)**: Clear, accurate answer.
- **🟦 (Expert)**: Can explain "Why" and provide edge cases.

## Post-Grading Actions
- Update `Mastery Tracker` in Dashboard.
- Update metadata in the specific concept note.
- If 🟥, re-queue concept for next session.
