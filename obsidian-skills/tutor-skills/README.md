# Tutor Skills for Obsidian

A comprehensive Socratic learning system for Gemini CLI, designed to transform source material into structured knowledge and track mastery.

## Features
- **Codebase Mode**: Automatically analyzes technical stacks and generates developer onboarding maps.
- **Phased Workflow**: D1-D9 execution strategy for consistent vault generation.
- **4 Quiz Modes**: Diagnostic, Drill (weakness focus), Section-based, and Hard-mode (deep understanding).
- **Mastery Tracker**: Visual proficiency tracking using Emoji badges (🟥/🟨/🟩/🟦).
- **Active Recall**: Built-in practice questions with folded answers.

## Components
- `tutor-setup`: The generator that creates the Obsidian StudyVault.
- `tutor`: The interactive Socratic mentor.

## Structure
- `references/`: Modular templates and rules for consistent output.

## Installation
1. Ensure you have the `obsidian-skills` directory in your vault.
2. Add these skills to your Gemini CLI `settings.json`.
