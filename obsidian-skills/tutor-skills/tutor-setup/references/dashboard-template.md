# Dashboard Template

## Overview
- **MOC (Map of Content)**: Central hub for all concepts.
- **Quick Reference**: Key commands and syntax.
- **Mastery Tracker**: Visual progress using badges (🟥/🟨/🟩/🟦/⬜).
