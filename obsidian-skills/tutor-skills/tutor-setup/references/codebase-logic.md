# Codebase Mode Logic

## Detection
- Identify tech stack: Check for `package.json`, `requirements.txt`, `go.mod`, `Cargo.toml`, `pyproject.toml`.
- Map entry points: Identify `main.py`, `index.ts`, `app.js`.

## Output Requirements
- **Architecture.md**: High-level flow using Mermaid.
- **DependencyMap.md**: Module relationships.
- **Onboarding.md**: Sequence of hands-on tasks.
- **DevSetup.md**: Instructions to get the project running.

## Rule
- Prioritize "How the data flows" over "What the files are".
