# Concept Note Format

## Structure
- **Definition**: Clear, concise explanation.
- **Context**: Why it matters.
- **Example**: Real-world or code snippet.
- **Active Recall**:
    > [!question]+ Practice Question
    > Question here...
    > > [!check]- Answer (Folded)
    > > Answer here...

## Linking
- Every note must link to at least 2 other related concepts.
